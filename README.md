# GD Landscapes Redesign

Static redesign prototype for GD Landscapes using local copies of the original site's content, imagery, testimonials and contact details.

## Files

- `index.html` - one-page brochure site
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and sticky header behaviour
- `assets/` - local image assets used by the page

Open `index.html` directly in a browser or serve the folder with any static web server.
